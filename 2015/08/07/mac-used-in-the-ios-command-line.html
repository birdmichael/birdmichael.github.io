Apache

启动本地的Apache服务器

sudo apachectl -k start

重新启动本地的Apache服务器

sudo apachectl -k restart

(涉及修改,就需要重新启动)

进入Apache配置文件目录

cd /etc/apache2/

打开Apache配置文件

sudo vim httpd.conf

Vim常用操作

vim中有两种模式
命令模式，用于输入命令(注意输入法要在英文状态)
    shift+V        可以选中一行
    y                    复制一行
    p                    在当前行下方粘贴复制的内容
    d$                删除到行尾
    x                    删除一个字符
    :wq                保存退出
    :q!                不保存退出
    i                    进入编辑模式
    /                   查找字符串

编辑模式：用于编辑按ESC可以切换回命令模式
PHP支持

载入PHP5模块

#LoadModule php5_module libexec/apache2/libphp5.so 中#去掉

cd /etc/apache2/
sudo vim httpd.conf
/php
i
(去掉#)
est(退出编辑模式)
wq (保存)

配置php.ini

将自带php.ini.default 拷贝为 php.ini

cd /etc/
sudo cp php.ini.default php.ini
